# 网站发布流程

## 准备数据

- 商品属性
    - sku
    - category
    - name
    - size: [length],[width],[height]
    - application: [app1],[app2]

- 商品照片
    - 图片切割
    - 图片优化
    - 放入文件夹(文件路径：category/skuNo/1.jpg)

- 首页大图
    - 每件热门商品都会有合适的大图么？

## 部署

- 代码压缩
    - 合并压缩js
    - 合并压缩css
    - 压缩html（去除注释和空行）

- 上传图片到指定文件夹

- 将数据导入指定表
    - 产品信息表
